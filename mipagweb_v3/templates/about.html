Proyecto mipagweb_v3

Base de datos

Se utilizaron bases de datos relacionales, con el gestor de bases de datos Sqlite3, en el cual se crearon 5 tablas con diferentes relaciones entre ellas, y dar mas interación a la pagina, datos adicionales que el usuario puede consultar.

Se muestra en el costado una imagen del las tablas y sus respectivas relaciones

Backend

Se utilizó el lenguaje python con el framework Django en su version 3.2.

Se utiliza el patron Modelo Vista Controlador. Se crearon los siguientes elementos para el desarrollo:

  • - Aplicación para gestion de base de datos
  • - Vistas con templates de Django
  • - Perfil administrativo para administrar los datos
  • - Modelo

Para mas detalles visitar el codigo fuente en GitHub

Frontend

Se usa Boostrap V4.3, aplicado directamente en los templates
